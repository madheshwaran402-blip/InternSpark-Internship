Task 2 – Synthesis & Gate-Level Simulation

Project

4-Bit Multiplier using Verilog HDL

Objective

To design a 4-Bit Multiplier, perform RTL simulation, synthesize the design using Xilinx Vivado, and analyze the generated hardware implementation.

Tools Used

* Xilinx Vivado 2025.2
* Verilog HDL
* XSim Simulator

Features

* 4-bit × 4-bit multiplication
* 8-bit product output
* Behavioral simulation
* Logic synthesis
* Hardware resource analysis

Design Flow

1. RTL Coding
2. Testbench Development
3. Behavioral Simulation
4. Logic Synthesis
5. Synthesized Schematic Analysis
6. Resource Utilization Analysis

Results

The 4-Bit Multiplier was successfully synthesized using Vivado. Behavioral simulation verified correct functionality, and the synthesized design demonstrated successful hardware implementation with efficient resource utilization.
